2.2 Biciclos

Se coloca el prefijo biciclo- antes del nombre del alcano correspondiente. Se  indica entre corchetes el número de átomos de carbono entre las cabezas de  puente (de mayor a menor).

Numeración: empiezapor una cabeza de puente → cadena más larga → cadena intermedia → más corta.

2.3 Espiranos

Compuestos bicíclicos donde un único C forma parte de ambos ciclos.

Nombrar: espiro[a.b]alcano (a,b = átomosde C a cada lado, menor a mayor).

Numerar: anillo pequeño → C espiránico → otro anillo.
