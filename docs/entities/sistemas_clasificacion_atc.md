# Sistemas de Clasificación y DCI

## ATC (Sistema de Clasificación Anatómica, Terapéutica y Química)
El código ATC es un sistema internacional jerárquico asignado a cada fármaco para organizar el inmenso vademécum mundial. Proporciona una "dirección anatómica" a la acción de un fármaco.

Su estructura consta de varios niveles alfanuméricos. Por ejemplo, el Diazepam es **N05BA01**:

1. **N** (Nivel 1): Sistema anatómico principal (*Sistema Nervioso Central*).
2. **05** (Nivel 2): Grupo terapéutico (*Agentes psicolépticos*).
3. **B** (Nivel 3): Subgrupo terapéutico (*Ansiolíticos e hipnóticos*).
4. **A** (Nivel 4): Grupo químico (*Derivados de benzodiazepina*).
5. **01** (Nivel 5): Sustancia química individual (*Diazepam*).

![Código ATC Diazepam](../images/imageFile41.png)

Existen grandes familias anatómicas base: (A) Alimentario/Tracto, (C) Cardiovascular, (N) Nervioso, (J) Antiinfecciosos, entre otros.

## DCI (Denominación Común Internacional / INN)
Ideadas por la Organización Mundial de la Salud (OMS), las DCI son los nombres "genéricos" de uso público sin propietario. Su propósito principal es conferir orden semántico: mediante prefijos o sufijos se define inmediatamente el clado de una molécula y su función general, favoreciendo la seguridad global del paciente y del médico.

**Reglas principales:**
No deben ser largas, deben ser fonéticamente fáciles de pronunciar y no hacer referencia a órganos anatómicos explícitos ni enfermedades.

**Ejemplos de Partículas Clínicas:**

| Partícula | Grupo Farmacológico |
|---|---|
| `-azepam` / `-azolam` | Benzodiazepinas clásicas |
| `-caina` | Anestésicos locales |
| `-dipina` | Antagonistas del calcio |
| `-micina` | Antibióticos derivados de *Streptomyces* |
| `-olol` | Betabloqueantes |
| `-pril` | Antihipertensivos IECA |
